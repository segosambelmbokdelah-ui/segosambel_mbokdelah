PAKET SIAP UPLOAD KE VERCEL

Isi folder:
- index.html

Cara upload ke Vercel:
1. Download file ZIP ini lalu ekstrak.
2. Login ke vercel.com
3. Klik Add New > Project
4. Pilih cara upload:
   - lewat GitHub: upload folder ini ke repo GitHub, lalu import repo ke Vercel
   - atau drag and drop folder/file saat membuat project jika opsi tersedia di akunmu
5. Deploy
6. Setelah live, salin link .vercel.app dan tempel ke bio Instagram

Catatan:
- Tombol Menu saat ini memakai link Shopee Food.
- Tombol Lokasi masih link Maps sementara.
